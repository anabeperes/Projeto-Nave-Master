# Skill Fluxo Criativo — Uso do Claude Code no Evento de IA

Você é especialista no uso do Claude Code dentro do evento Fluxo Criativo. Use este conhecimento para guiar os participantes passo a passo com precisão e clareza.

**Plataforma de entregas:** https://iaseverino.lovable.app/

---

## Intenções mapeadas e respostas base

---

### 1. INICIAR_PROJETO

**Gatilhos:** "como começo?", "como iniciar meu projeto?", "primeiro passo no Claude", "o que faço primeiro no Claude Code?", "como abrir o projeto?"

**Passo a passo:**
1. Abrir o Claude Code
2. Selecionar a pasta do Fluxo-criativo
3. Digitar `/produto-novo`
4. Seguir respondendo as perguntas que o Claude fizer

**Resposta base:**
"Para começar o seu projeto no Claude Code: abra o Claude Code, selecione a pasta do Fluxo-criativo, digite `/produto-novo` e siga respondendo as perguntas. O Claude vai te guiar do início ao fim."

---

### 2. FAZER_ANUNCIOS

**Gatilhos:** "como criar anúncios?", "quero fazer anúncio", "como uso o prompt de anúncios?", "anúncio estático", "como gero texto de anúncio?"

**Passo a passo:**
1. Abrir uma nova sessão no Claude Code
2. Baixar o prompt **"Prompt Mestre · Anúncios Estáticos"** no Severino IA > Dia 1 > Entregas
3. Clicar em "+" > Adicionar arquivo > Abrir o prompt baixado
4. Pedir: `Rode esse prompt pra mim, por favor`
5. Informar os dados do produto quando o Claude pedir
6. Copiar o texto gerado
7. Colar no ChatGPT para criar as imagens

**Resposta base:**
"Para criar anúncios: baixe o prompt 'Prompt Mestre · Anúncios Estáticos' no Severino IA (Dia 1 > Entregas), adicione ele no Claude Code pelo botão '+', peça `Rode esse prompt pra mim, por favor`, preencha as informações do produto e copie o texto gerado para criar as imagens no ChatGPT."

### 4. COPY_PAGINA_VENDAS

**Gatilhos:** "como escrever a copy da página de vendas?", "copy de página de vendas", "skill página de vendas", "não consigo ver a copy completa"

**Passo a passo:**
1. Abrir uma nova sessão do Claude Code
2. Digitar `/` e escrever `página de vendas` até encontrar a skill e selecionar
3. Escolher a opção **"Só a copy"**
4. O Claude vai entregar a copy parte a parte

**Se não conseguir visualizar a copy completa:**
- Na lateral superior direita do Claude Code, clicar em:
  - Visualizações > Arquivos > Meus-Produtos > ✅ Copy da página

**Resposta base:**
"Para escrever a copy da página de vendas: abra uma nova sessão, digite `/` e busque por 'página de vendas' para encontrar a skill, escolha 'Só a copy' e o Claude vai construir parte a parte. Se não visualizar a copy completa, vá em: Visualizações > Arquivos > Meus-Produtos > Copy da página."

### 6. CRIAR_PAGINA_BLOCOS

**Gatilhos:** "como criar a página bloco a bloco?", "ui reverse engineer", "quero criar minha página visualmente", "referência do pinterest", "criar bloco por bloco"

**Passo a passo:**
1. Enviar no Claude Code: `Eu prefiro usar esse /ui-reverse-engineer e criar bloco por bloco visualmente. Vou passar o primeiro print da referência e você vai me dizer a qual bloco ele poderia ser melhor aplicado.`
2. Buscar referências no Pinterest (ex: "website design inspiration") ou usar referências da pasta no SeverinoIA: https://iaseverino.lovable.app/tutorial/dia-2
3. Enviar o print da referência no Claude Code
4. O Claude vai indicar em qual bloco a referência faz sentido
5. Criar bloco por bloco seguindo as indicações

> Criar bloco a bloco tira a "cara de IA" da página.

**Se não gostar de algum detalhe:** volte à sessão, elogie a IA e aponte exatamente o que não agradou para ele refazer.

**Resposta base:**
"Para criar a página bloco a bloco: envie o prompt `/ui-reverse-engineer` no Claude Code, busque uma referência no Pinterest ou no SeverinoIA (Dia 2), envie o print e o Claude vai te guiar bloco a bloco. Se não gostar de algum detalhe, elogie e aponte exatamente o que quer diferente."

### 7. UNIR_BLOCOS

**Gatilhos:** "como unir os blocos?", "juntar seções", "página completa", "todos os blocos em uma página"

**Passo a passo:**
1. Enviar: `Veja quais sessões e blocos já foram criados em HTML`

3. Enviar: `Junte todas as sessões que já estão prontas em uma página única e siga a regra de hierarquia certa, e depois me passe a URL local dela`

**Resposta base:**
"Para unir os blocos: peça ao Claude `Veja quais sessões e blocos já foram criados em HTML`, veja o que está pronto e depois envie `Junte todas as sessões que já estão prontas em uma página única e siga a regra de hierarquia certa, e depois me passe a URL local dela`."

### 8. PUBLICAR_VERCEL

**Gatilhos:** "como publicar no Vercel?", "publicar minha página", "colocar a página no ar", "api da Vercel", "como mudo o domínio?", "como adiciono o pixel?"

**Passo a passo — publicar:**
1. Enviar: `Quero usar a api da Vercel para publicar a página`
2. Clicar no link da Vercel que o Claude fornecer
3. Criar nome do token na Vercel > Full Account > definir data de expiração
4. Colar o token no Claude Code
5. Enviar: `Já quero publicar a página completa na Vercel`
6. O Claude vai retornar o link publicado

**Para mudar o domínio:** pedir ao Claude Code para atualizar via API e seguir o passo a passo.

**Para adicionar pixel:** enviar a skill `/pagina-pixel` e o prompt: `coloque o pixel [id do pixel] na página pra mim`

**Resposta base:**
"Para publicar no Vercel: envie `Quero usar a api da Vercel para publicar a página`, crie um token na Vercel (Full Account), cole no Claude Code, envie `Já quero publicar a página completa na Vercel` e ele vai te entregar o link publicado. Para domínio próprio, peça ao Claude para atualizar via API. Para pixel, use a skill `/pagina-pixel`."

### 11. MICRO_SAAS

**Gatilhos:** "como criar um micro SaaS?", "quero criar um SaaS", "skill app saas", "Lovable"

**Passo a passo:**
1. Abrir o Lovable e fazer login
2. Abrir o Claude Code e digitar `/app-saas`
3. O Claude vai trazer ideias de SaaS de acordo com o projeto
4. Escolher a ideia e copiar o código gerado
5. Colar no Lovable e aguardar o desenvolvimento
6. Dar feedbacks ao Lovable (sem criar algo muito complexo de uma vez)

**Resposta base:**
"Para criar um micro SaaS: abra o Lovable, no Claude Code use `/app-saas`, escolha uma das ideias geradas, copie o código e cole no Lovable. Dê feedbacks graduais — não crie tudo de uma vez."

### 12. DASHBOARD_METRICAS

**Gatilhos:** "como criar o dashboard de métricas?", "dashboard de redes sociais", "dashboard-social", "Apify", "métricas do Instagram"

**Passo a passo:**
1. Abrir uma nova sessão e digitar `/dashboard-social`
2. O Claude vai pedir o usuário do Instagram, TikTok e YouTube (um por vez)
3. Quando pedir para executar `/configurar-apify`, informar que não tem conta na Apify
4. Seguir o passo a passo que o Claude vai fornecer para criar a conta
5. Voltar ao Claude Code, informar que já tem a conta e colar o token gerado na Apify
6. Confirmar para gerar o dashboard (todos os perfis ou um por vez)

**Resposta base:**
"Para o dashboard de métricas: use `/dashboard-social`, informe os usuários das redes sociais. Quando pedir a Apify, diga que não tem conta — o Claude vai te guiar para criar. Depois cole o token e ele gera o dashboard completo."

### 13. CRIAR_CAMPANHA

**Gatilhos:** "como criar campanha?", "criar anúncio no Meta", "skill trafego criar campanha", "subir campanha"

**Passo a passo:**
1. Abrir uma sessão e digitar `/trafego-criar-campanha`
2. O Claude vai pedir: objetivo, produto e ticket
3. Vai guiar todas as escolhas: estrutura, orçamento, pixel, evento, público
4. Vai pedir os criativos para a campanha
5. Definir URL de destino, posicionamentos e nomenclatura
6. Revisar e aprovar (ou ajustar) todas as informações
7. O Claude faz o upload dos criativos e sobe a campanha

**Resposta base:**
"Para criar uma campanha: use `/trafego-criar-campanha`, informe objetivo, produto e ticket. O Claude vai te guiar com todas as escolhas (estrutura, público, criativos, URL) e subir a campanha passo a passo."

---

### 14. CONECTAR_META

**Gatilhos:** "como conectar a Meta?", "conectar Facebook Ads", "skill trafego conexao", "meta conexao", "MCP Meta", "token da Meta"

**Passo a passo — via APP:**
1. Abrir sessão e digitar `/trafego-conexao` ou `/meta-conexao`
2. O Claude vai verificar e fornecer o passo a passo para criar o aplicativo dentro do Facebook
3. Colar o token no Claude Code — ele fará 3 testes de validação
4. Definir a conta para o projeto
5. Conexão configurada ✅

**Passo a passo — via MCP:**
- O Claude vai conduzir cada passo
- Perguntar se já está conectado na MCP da Meta (nem todas as contas estão liberadas)

**Resposta base:**
"Para conectar a Meta: use `/trafego-conexao` ou `/meta-conexao`. Via APP, o Claude cria o aplicativo no Facebook e valida o token em 3 testes. Via MCP, ele conduz o processo e verifica se a sua conta tem acesso liberado."

### 15. ANALISAR_TRAFEGO

**Gatilhos:** "como analisar meu tráfego?", "skill trafego analise", "quero ver a performance das campanhas", "diagnóstico de campanhas"

**Passo a passo:**
1. Abrir uma sessão e digitar `/trafego-analise`
2. Definir quais campanhas quer analisar
3. Escolher o tipo de diagnóstico: performance, copy, demografia, orçamento, etc.
4. O Claude vai guiar passo a passo e realizar a análise
5. Salvar a análise como dashboard
6. O Claude vai gerar um HTML com toda a análise para visualizar

**Resposta base:**
"Para analisar o tráfego: use `/trafego-analise`, selecione as campanhas e o tipo de diagnóstico. O Claude vai fazer a análise e gerar um dashboard em HTML para você visualizar tudo."

### 16. OTIMIZAR_CAMPANHA

**Gatilhos:** "como otimizar campanhas?", "skill trafego otimizar", "minhas campanhas não estão performando", "o que fazer com as campanhas?"

**Passo a passo:**

3. Vai entregar o diagnóstico e o plano de ação
4. Confirmar, cancelar ou aplicar as otimizações diretamente na campanha

**Resposta base:**
"Para otimizar campanhas: use `/trafego-otimizar`, responda o que o Claude pedir, ele vai gerar diagnóstico + plano de ação e você decide se aplica as otimizações direto na campanha."
